Результат:
Техническое задание на разработку программы "Система управления библиотекой"
Введение
Программа предназначена для автоматизации управления библиотекой. 
Система должна предоставлять возможность добавления, удаления и поиска книг,
а также управления пользователями и их заимствованиями.

Основные концепции ООП
Инкапсуляция: Скрытие деталей реализации через использование модификаторов доступа.
Наследование: Использование наследования для создания специализированных классов на основе базовых.
Полиморфизм: Реализация полиморфного поведения через переопределение методов.
Абстракция: Выделение общих характеристик объектов и представление их в виде абстрактных классов и интерфейсов.

Основные классы
Класс Book

Свойства: Title, Author, ISBN, PublicationYear
Методы: DisplayInfo()
Класс User

Свойства: UserID, Name, Email
Методы: DisplayInfo()
Класс Borrowing

Свойства: BorrowingID, Book, User, BorrowDate, ReturnDate
Методы: DisplayInfo()
Класс Library

Свойства: Books (список книг), Users (список пользователей), Borrowings (список заимствований)
Методы: AddBook(), RemoveBook(), FindBook(), AddUser(), RemoveUser(), FindUser(), RegisterBorrowing(), ReturnBook()

Поиск книг и пользователей.
Возврат книги.
Требования к программе
Язык программирования: C#
Среда разработки: Visual Studio или любая другая поддерживающая C#
Хранение данных: В памяти программы (реализация в виде коллекций)
Интерфейс: Консольное приложение
Дополнительные требования
Обработка исключений при некорректном вводе данных.
Возможность расширения функционала без значительных изменений в существующем коде.
Логирование действий (например, добавление книги, заимствование и возврат).
Заключение
Разработанная программа должна быть объектно-ориентированной, легко расширяемой и поддерживаемой. 
Использование основных принципов ООП обеспечит высокую степень модульности и повторного использования кода.
